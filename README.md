
# WP Podmind Chat

A WordPress plugin to use voxgig podcast Chat.

## Local test:
- Zip the source files. For example:
```bash
cd wp_podmind_chat
zip wp_podmind_chat.zip *
```
- Upload, install and activate the plugin:

- Add the API key:
